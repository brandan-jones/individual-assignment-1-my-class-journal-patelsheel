My Journal Entry Project

Hello, world!

Welcome to my project. This project has the following tasks:

  • Add an index.html page (or start.html if you wish to map to /start).
  • Create a DTO. Use Lombok annotations instead of creating getter and setter methods.
  • Create a service class. This class should:
    • Have an @Service annotation.
    • Implement an interface that has been provided for you.
    • When the save method is executed, store the journal entry in a collection of your choice (ArrayList, HashMap, etc.).
    • When the fetchAll method is executed, return all journal entries.
  • Add CircleCI to your project. I've provided a yaml file. You won't need to make any changes to this configuration; simply register your repo with CircleCI and ensure all tests pass.
  • Do something extra, something not listed here.
